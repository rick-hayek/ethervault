# EtherVault - ローカルパスワード管理ツール

EtherVault は、頑強なセキュリティとシームレスなユーザー体験を提供するように設計された、モダンなマルチプラットフォーム対応のパスワードマネージャーです。デスクトップおよびモバイル (Android/iOS) プラットフォームをサポートし、機密性の高い資格情報の安全な保存、管理、同期を支援します。

[English](./README.md) | [中文](./README.zh.md) | 日本語 | [한국어](./README.ko.md)

## 📖 プロジェクト概要

EtherVault は Monorepo 構成を採用し、コアとなるビジネスロジックを UI レイヤーから分離することで、クロスプラットフォームにおける高いコード再利用性と一貫性を実現しています。強力なローカル暗号化 (AES-256) を内蔵し、Google ドライブなどの主要なクラウドストレージサービスとのエンドツーエンドの暗号化同期をサポートしているほか、生体認証によるロック解除やパスワードの健全性分析などの高度な機能を備えています。ユーザーにとって最も安全なデジタル金庫を提供することを目指しています。

## 🚀 主な機能

### 🔐 コア保管庫
- **資格情報管理**: ユーザー名、パスワード、URL、メモを安全に保管します。
- **効率的な検索**: あいまい検索、カテゴリフィルタリング (すべて、個人用、仕事用、その他)、および多次元タグ管理をサポートします。
- **スマートアイコン**: ウェブサイトのアイコンを自動的に取得して表示し、より良い視覚体験を提供します。

### 🛡️ セキュリティダッシュボード
- **セキュリティスコア**: パスワードの複雑さのアルゴリズムに基づき、保管庫の健全性をリアルタイムで評価します。
- **リスク検出**: 脆弱なパスワードや使い回されているパスワードを自動的にスキャンし、最適化の提案を行います。
- **データの視覚化**: 直感的なチャートでパスワードのセキュリティ分布を視覚化します。

### 🎲 パスワードジェネレーター
- **高いエントロピー**: 推測不可能な強力なパスワードを生成します。
- **高いカスタマイズ性**: カスタムの長さ (最大 128 文字) や文字セット (大文字、小文字、数字、記号) をサポートします。
- **ワンクリックコピー**: 自動的にエントロピーを計算し、生成後にワンクリックでコピーできます。

### ⚙️ 設定とエコシステム
- **クラウド同期**: Google ドライブなどのクラウドプロバイダーと暗号化されたデータの同期をサポートし、デバイス間の一貫性を保証します。
- **テーマ**: ダークモード、ライトモード、システムデフォルトをサポートします。
- **多言語対応**: 内蔵の多言語サポート (英語/中国語、さらに拡大中)。
- **データの移行**: 簡単なデータ移行のための標準的な CSV/JSON インポートおよびエクスポートをサポートします。
- **セキュリティ保護**: 生体認証によるロック解除 (FaceID/TouchID)、自動ロック、およびローカル操作ログの監査をサポートします。

## 💻 開発ガイド

### 前提条件
- Node.js (v18 以上推奨)
- NPM

### 1. Google Console のセットアップ (クラウド同期用)
アプリを実行する前に、Google ドライブ同期を有効にするために Google Cloud Console でプロジェクトをセットアップする必要があります。

1.  [Google Cloud Console](https://console.cloud.google.com/) にアクセスします。
2.  新しいプロジェクトを作成します (例: `ethervault-dev`)。
3.  **API の有効化**:
    *   **API とサービス > ライブラリ** に移動します。
    *   **Google Drive API** を検索して有効にします。
4.  **OAuth 同意画面の設定**:
    *   **API とサービス > OAuth 同意画面** に移動します。
    *   **外部 (External)** を選択します。
    *   必要なアプリ情報を入力します。
    *   **テストユーザー** として自分のメールアドレスを追加します。
5.  **認証情報の作成 (Web クライアント)**:
    *   **API とサービス > 認証情報** に移動します。
    *   **認証情報を作成 > OAuth クライアント ID** をクリックします。
    *   アプリケーションの種類: **ウェブ アプリケーション**。
    *   承認された JavaScript 生成元を追加: `http://localhost:3000` (必要に応じて `http://localhost:5173`)。
    *   承認されたリダイレクト URI を追加: `http://localhost:3000` (必要に応じて `http://localhost:5173`)。
    *   **クライアント ID** をコピーします。
6.  **認証情報の作成 (iOS クライアント)**:
    *   **API とサービス > 認証情報** に移動します。
    *   **認証情報を作成 > OAuth クライアント ID** をクリックします。
    *   アプリケーションの種類: **iOS**。
    *   Bundle ID を追加: `com.ethervault.app`。
    *   **クライアント ID** をコピーします。
7.  **環境変数の設定**:
    *   `packages/app/.env` ファイルを作成します (`.env.example` からコピー)。
    *   認証情報を追加します:
        ```bash
        # ローカル開発用
        VITE_GOOGLE_CLIENT_ID=your_client_id_here
        # iOS、Android、デスクトップのネイティブアプリ用
        VITE_GOOGLE_CLIENT_ID_IOS=your_client_id_here
        ```

### 2. 依存関係のインストール
プロジェクトのルートディレクトリで実行します:
```bash
npm install
```

### 3. 開発環境の起動

**Web モード (ブラウザ):**
```bash
npm run dev
```

**デスクトップモード (Electron):**
```bash
npm run dev:desktop
```

**モバイル同期 (Capacitor):**
```bash
npm run mobile:sync
```

### 4. ビルドとパッケージング

**Web アセットのビルド:**
```bash
npm run build
```

**デスクトップアプリのビルド:**
```bash
npm run dist:desktop
```

**iOS/Android のビルド:**
```bash
# iOS
npm run build:ios

# Android
npm run build:android
```

### 5. その他のコマンド

**プロジェクトのクリーン:**
```bash
npm run clean
```

**型チェック:**
```bash
npm run type-check
```
